## How to upload on GitHub website

1. Open your GitHub repository.
2. Click **Add file** → **Upload files**.
3. Drag the folders from this patch into the upload area:
   - `notebooks`
   - `src`
   - `scripts`
   - `paper`
4. GitHub will merge these into the existing folders.
5. Use this commit message:

```text
Add missing starter files and paper source
```

6. Click **Commit changes**.
